# node-brouser

A lightweight web browser built with [Node.js](https://nodejs.org).

## About

`node-brouser` is a browser built from scratch using Node.js. This project explores how browsers work under the hood — rendering, navigation, and everything in between.

## Status

🚧 **Early development** — core features are being built.

## Planned Features

- Web page rendering
- URL navigation
- Tab support
- History tracking
- Bookmarks

## Tech Stack

- [Node.js](https://nodejs.org)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) v18 or higher
- npm (comes with Node.js)

```bash
git clone https://github.com/brycethayer1323/node-brouser.git
